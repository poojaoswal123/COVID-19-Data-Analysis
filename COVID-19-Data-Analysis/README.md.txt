# 🦠 COVID-19 Data Analysis using Python

## 📌 Project Overview

This project analyzes the COVID-19 Time Series dataset to identify trends in confirmed cases, recoveries, deaths, and active cases across different countries. The project includes data cleaning, exploratory data analysis (EDA), data visualization, and business insights using Python.

The objective is to understand the global impact of COVID-19 through meaningful visualizations and statistical analysis.

---

## 📂 Dataset

- **Source:** Kaggle
- **Dataset:** COVID-19 Time Series Dataset
- **File Used:** `time-series-19-covid-combined.csv`

---

## 🎯 Project Objectives

- Perform data cleaning and preprocessing.
- Handle missing values and duplicate records.
- Analyze confirmed, recovered, death, and active cases.
- Identify trends over time.
- Compare countries based on COVID-19 statistics.
- Generate business insights using data visualization.

## ▶️ How to Run the Project

1. Clone the repository.

```bash
git clone https://github.com/your-username/COVID-19-Data-Analysis.git
```

2. Navigate to the project folder.

```bash
cd COVID-19-Data-Analysis
```

3. Install the required libraries.

```bash
pip install -r requirements.txt
```

4. Open the notebook in Google Colab or Jupyter Notebook.

5. Run all cells to generate the visualizations and business insights.
